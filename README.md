## 🚗 Car Racing Game

A simple Car Racing Game developed using **C++**. The player controls a car, avoids enemy vehicles, and tries to achieve the highest score without crashing.

---

## 🎮 Features

- 🚗 Player-controlled car
- 🚙 Random enemy cars
- ⌨️ Keyboard controls
- 💥 Collision detection
- 📈 Score system
- 🔄 Restart option
- 🎯 Increasing difficulty

---

## 🛠️ Technologies Used

- Code::Blocks / Visual Studio / Dev-C++ (যেটা ব্যবহার করেছো)

## 🎮 Controls

| Key | Action |
|-----|--------|
| ⬅ Left Arrow | Move Left |
| ➡ Right Arrow | Move Right |
| Esc | Exit Game |
